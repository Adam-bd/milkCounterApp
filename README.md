# 🍼 Milk Counter App

Aplikacja do monitorowania i zliczania zapasów odciągniętego mleka matki.

## 🌟 O projekcie
**Milk Counter App** to prosta i intuicyjna aplikacja desktopowa stworzona z myślą o młodych rodzicach. Pomaga w organizacji, śledzeniu i zarządzaniu zapasami odciągniętego mleka. 

Dzięki temu narzędziu możesz w łatwy sposób kontrolować dostępne porcje oraz śledzić historię wagi malucha, aby lepiej planować karmienie.

### ✨ Główne funkcje
* **Zarządzanie magazynem:** Dodawanie i zużywanie porcji mleka.
* **Śledzenie wagi:** Monitorowanie historii wagi dziecka.
* **Personalizacja:** Konfiguracja ustawień aplikacji pod własne potrzeby.

## 🛠️ Technologie
* **Język:** Java 
* **Stylizowanie interfejsu:** CSS (JavaFX)
* **Zarządzanie zależnościami i budowanie:** Maven
* **Przechowywanie danych:** pliki CSV zapisywane lokalnie na komputerze
